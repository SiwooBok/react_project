# 장바구니 기능 추가중

그동안 마크다운 문서를 작성하는 것을 일부 누락해서 여기에 정리하고자 한다.

이제까지 만든 것은 다음과 같다.
- 메인페이지
- 서브페이지(About, Shop, Cart, 404 Page 등)
- 경고창 컴포넌트
- Shop.js 내부의, ShopPopup 컴포넌트. (상품 클릭시 모달창을 보여준다.)

이제 장바구니에 추가할 수 있도록 리덕스를 이용해 기능을 구현해보고자 한다.
